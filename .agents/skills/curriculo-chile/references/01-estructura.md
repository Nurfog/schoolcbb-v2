# Estructura del Sistema Educativo Chileno

## Niveles y Codificación MINEDUC

| Nivel | Cursos | Codificación |
|-------|--------|-------------|
| Educación Parvularia | Sala Cuna (SC), Nivel Medio (NM), Nivel Transición (NT) | SC, NM, NT |
| Educación Básica (Ciclo I) | 1° a 6° Básico | PB01–PB06 |
| Educación Básica (Ciclo II) | 7° a 8° Básico | PB07–PB08 |
| Educación Media HC | 1° a 2° FG + 3° a 4° Diferenciado HC | MM01–MM04 |
| Educación Media TP | 3° a 4° FG + Especialidad TP | TP03–TP04 |
| Educación Media Artística | 3° a 4° FG + Diferenciado Artístico | AR03–AR04 |
| EPJA | Educación de Personas Jóvenes y Adultas | EPJA |

## Tipos de Enseñanza
- **HC:** Humanista-Científico
- **TP:** Técnico-Profesional
- **CH:** Artística

## Dependencia Administrativa (DELE)
- 1 = Municipal
- 2 = PSU-SLE
- 3 = Particular Subvencionado
- 4 = Particular Pagado

## RBD
Rol Base de Datos del establecimiento. Código único de 6 dígitos asignado por MINEDUC.
